    # 将两个元素设置为display:inline-block时会有诡异的差距

    将两个或多个元素设置成 display: inline-block 或 display: inline 会让每两个元素间创建一个小空白。这个小空白是由于浏览器将这些元素解析成字，在它们之间添加一个字符的空白。 一个简单的解决方法，是对父元素设置 font-size: 0，再单独对子元素设置字体大小。

    ul {
    font-size: 0;
}

li {
    font-size: 16px;
}
    # label与input之间的关系

    给input设置Id,给Label设置for，for的内容为input的id，当我们点击label时，input也会聚焦，此方式可以用在input为check、radio时很好用。

    <label for="emailAddress">Email address:</label>
<input type="email" id="emailAddress">
