    # 兼容性问题

    1、iphone的300毫秒延迟 可以使用引入fastclick解决 iphone的scroll会有延迟卡顿的问题，加上一个样式即可解决 -webkit-overflow-scrolling:touch;

    2、样式兼容问题:padding与margin

    3、移动端手机设置100vh和100%的区别，100vh在safiri会计算到底部的上一栏的高度，而100%不会出现这种问题

    4、video自动播放，一种是引入vue-player-video实现，但是会有浏览器限制，如谷歌需要设置一个自动播放才可以出现，相反，原生的可以实现自动播放，但需要设置静音播放才可以实现。

    5、微信小程序的坑点有2 a、textarea的层级问题； 我的解决思路，建立一个一样大小文字一样的，进行if判断，当点击这个框的时候，然后调聚焦方式，当失去焦点的时候if等于false，显示另外一个固定好的输入框。

    6、获取对象长度的方法

    for (let property in res.data.data){
    if(Object.prototype.hasOwnProperty.call(res.data.data,property)){
        count++
    }
}

    function permissionForShake() {
    if (typeof DeviceMotionEvent.requestPermission === 'function') {
        DeviceMotionEvent.requestPermission()
            .then(permissionState => {
                if (permissionState === 'granted') {
                    window.addEventListener('devicemotion', () => {})
                }
            })
            .catch(console.error);
    } else {
        // ios其他系统可以不通过请求直接摇一摇
        
    }
}

    10、部分安卓机型做动画操作时css请求的是第三方资源，会有以下报错

     DOMException: Failed to execute 'insertRule' on 'CSSStyleSheet': Cannot access StyleSheet to insertRule

    该问题可能是资源加载的问题，即，将第三方css资源移动至body标签最底部即可。
